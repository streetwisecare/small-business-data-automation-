# Presentation Outline

## 1. What I Built
- A small data-automation system for a landscaping business.
- It organizes exported business files and creates daily backups.

## 2. Website Overview
- Shows jobs, expenses, and reports.
- Can export data into CSV/JSON files.

## 3. Automation Overview
- Automatically sorts files into dated folders.
- Creates daily backups.
- Generates file integrity hashes.
- Deletes old backups to stay organized.

## 4. Why This Matters
- Prevents data loss.
- Keeps business information clean and easy to find.
- Helps with digital forensics by preserving history and integrity.

## 5. Results
- Backup system works correctly.
- Restore process was tested successfully.
- All files validated with matching hashes.

## 6. Conclusion
- Simple system that makes managing a small business easier.
- Reliable, organized, and easy to maintain.
